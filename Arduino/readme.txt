tft folder contains the arduino project using hte 1.8" tft display from adafruit
this version requires a arduino mega and can be configured over the serial link
you do not need to edit teh source code at all just upload as is.  

config.exe will register MSCOMM32.OCX on first run if you dont have it.
If you have a problem make sure to run as admin once

\config\arduino contains a test sketch to debug the config code independantly.
